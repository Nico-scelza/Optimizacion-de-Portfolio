# Optimizacion-de-Portfolio
Optimizador de Portafolios de Inversión "Quantamental" desarrollado en Python. Implementa la Teoría Moderna de Carteras (Markowitz) ajustando el modelo CAPM con el Riesgo País Argentino en tiempo real y permitiendo inputs fundamentales manuales. Incluye simulación de Monte Carlo y Frontera Eficiente interactiva.
